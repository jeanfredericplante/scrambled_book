# Scrambled Sentence Game Generator

A modern web application to generate scrambled sentence worksheets for children to practice handwriting and spelling.

**[Live Demo](#)** *(Once published to GitHub Pages, the link will be here)*

## Features

- **Add Custom Sentences**: Create your own list of sentences.
- **Auto-Scramble**: Words are automatically shuffled.
- **Image Support**: Upload your own images or easily search for relevant ones on Google Images.
- **PDF Export**: Generate a printable PDF with images, scrambled text, and handwriting lines.
- **Privacy Focused**: All processing happens in your browser. No data is sent to any server.

## How to Use

1. **Add a Sentence**: Type the sentence you want the child to unscramble.
2. **Add an Image** (Optional):
   - Click "Find image on Google" to search for a related picture.
   - Save the image to your device.
   - Click "Choose File" to upload it to the card.
3. **Repeat**: Add as many sentences as you like.
4. **Download**: Click "Download PDF" to get your worksheet.

## Publishing to GitHub Pages

To make this application live on the web:

1. Go to your repository **Settings**.
2. Navigate to the **Pages** section (on the left sidebar).
3. Under **Source**, select `Deploy from a branch`.
4. Select the `main` (or `master`) branch and the `/ (root)` folder.
5. Click **Save**.

Your site will be ready at `https://<your-username>.github.io/<repo-name>/` in a few minutes.
